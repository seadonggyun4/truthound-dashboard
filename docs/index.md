# Truthound Dashboard

Truthound Dashboard is the operational interface for Truthound 3.0. The 3.0
dashboard keeps the existing visual shell and Intlayer-based application i18n,
but it no longer invents dashboard-only validation products or mock execution
paths.

## Scope

- Truthound 3.0 backed workflows only
- Source, validation, profile, drift, privacy, checkpoint, reporter, plugin,
  lineage, observability, storage, anomaly, and enterprise-sampling surfaces
- Documentation that maps dashboard behavior to canonical Truthound APIs

## Removed from the 3.0 surface

- Glossary, catalog, activity, and collaboration
- Standalone maintenance product pages
- Standalone drift monitoring, schema watcher, and model monitoring products
- Mock/demo mode, fallback backends, and phase-based compatibility messaging
